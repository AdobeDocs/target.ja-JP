トレーニングビデオでは Target Standard と Target Premium を最大限に活用する方法を説明しています。

次のカテゴリのビデオが視聴できます。

* Experience Cloud および全般機能
* アクティビティの作成
* オーディエンスとターゲティング
* アセット管理
* 設定と環境設定
* 実装とトラブルシューティング
* マーケティングの使用例
