# コンテキスト属性の作成 {#creating-your-context-attributes}

コンテキスト属性は、ユーザー、セッション、アプリケーションコンテキスト（サブスクリプション層、アプリバージョン、地域など）を記述するカスタムデータフィールドです。 コンテキスト属性を使用して、機能フラグのオーディエンス基準を定義します。

コンテキストグループは、関連するコンテキスト属性を論理階層に整理します。 コンテキストグループを使用すると、機能の設定中に属性を簡単に見つけて管理できます。

| 用語 | 説明 | 使用中 |
| --- | --- | --- |
| **コンテキスト属性** | データタイプとオプションの定義済み値を持つ名前付きフィールド。 | オーディエンス基準、機能設定 |
| **コンテキストグループ** | 関連するコンテキスト属性を整理するカテゴリ。 | フィーチャ作成時のコンテキスト選択 |
| **UI ラベル** | ユーザーインターフェイスに表示される表示名。 | オーディエンス基準と管理ページ |
| **ID** | 固有の技術識別子。 | アプリケーションのリクエスト |
| **定義済みの値** | 表示ラベルを含む許可される値のオプションのリスト。 | ドロップダウンリストとオーディエンスルールビルダー |

## 前提条件 {#prerequisites}

コンテキスト属性とコンテキストグループを管理する前に、次の手順を実行します。

* **フラグコンソール**&#x200B;に正しいサンドボックスでアクセスできます。
* コンテキスト属性とコンテキストグループの作成と管理に必要な&#x200B;**管理者**&#x200B;の役割があります。

## コンテキスト属性に移動 {#navigate}

1. **フラグ コンソール**&#x200B;にログインします。
1. 左側のナビゲーションの&#x200B;**ダッシュボード**&#x200B;で、**コンテキスト属性**&#x200B;を選択します。

現在、**コンテキスト属性** ページにアクセスしています。 このページを使用して、コンテキスト・グループとコンテキスト属性を作成します。

## コンテキストグループの作成 {#create-group}

すべてのコンテキスト属性はグループに属する必要があります。

1. 左側のナビゲーションの&#x200B;**ダッシュボード**&#x200B;で、**コンテキスト属性**&#x200B;を選択します。
1. 「**My Context Attributes**」タブで、「**グループを追加**」を選択します。
1. 必要なフィールドに情報を入力します。
1. **作成**&#x200B;を選択します。

### グループフィールド {#group-fields}

| フィールド | 説明 |
| --- | --- |
| **グループ名** | グループの一意の名前を入力します。 |
| ****&#x200B;のサブグループを作成 | 階層を整理する親グループを選択します。 属性の作成中にグループを作成することもできます。 |

## コンテキスト属性の作成 {#create-attribute}

1. **My Context Attributes** タブで、**New Context Attribute**&#x200B;を選択します。
1. 必要なフィールドに情報を入力します。
1. **送信**&#x200B;を選択します。

### 属性フィールド {#attribute-fields}

| フィールド | 説明 |
| --- | --- |
| **UI ラベル** | オーディエンス条件で使用される表示名を入力します。 |
| **ID** | アプリケーション要求で使用される一意の技術識別子を入力します。 |
| **コンテキスト グループ** | グループを選択するか、インラインで新しいグループを作成します。 |
| **データタイプ** | アプリケーションが送信するデータに一致するタイプを選択します。 |

| データタイプ | 例 |
| --- | --- |
| 文字列 | `gold` |
| 整数 | `42` |
| ブーリアン | `true` |
| 小数 | `9.99` |
| 日付 | `2026-07-17` |
| バージョン | `1.2.0` |

### 定義済みの値（オプション） {#predefined-values}

事前定義済みの値を使用して、属性を固定の値セットに制限します。 「**定義済み値を追加**」を選択します。

| フィールド | 説明 |
| --- | --- |
| **UI ラベル** | 表示名を入力します。 |
| **値** | アプリケーションから送信された技術値を入力します。 |

**保存**&#x200B;を選択して、変更を適用します。

## トラブルシューティング {#troubleshooting}

| 問題 | 解決策 |
| --- | --- |
| **Create**&#x200B;は無効です | **グループ名**&#x200B;と&#x200B;**の両方を完了し、**&#x200B;のサブグループを作成します。 |
| 属性を送信できません | **ID**&#x200B;および&#x200B;**コンテキストグループ**&#x200B;を完了します。 |
| ドロップダウンにグループが表示されない | ページを更新するか、属性フォームからグループをインラインで作成します。 |
| 定義済みの値は保存されません | 属性を送信する前に、モーダルで「**保存**」を選択します。 |
