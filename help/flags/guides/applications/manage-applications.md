# アプリケーションの管理 {#manage-applications}

フラグの&#x200B;**アプリケーション**&#x200B;は、機能フラグで制御するサービスまたは製品を表します。 機能フラグを作成する前に、少なくとも1つのアプリケーションをコンソールにオンボーディングする必要があります。

## アプリケーションを管理できるユーザー {#permissions}

**管理者**&#x200B;の役割を持つメンバーのみがアプリケーションを追加または変更できます。 この役割がない場合は、管理者にお問い合わせください。

## アプリケーションの追加 {#add-application}

アプリケーションを追加する手順については、[&#x200B; アプリケーションのオンボーディング &#x200B;](onboard-your-application.md)を参照してください。
