# レコメンデーションアクティビティ

Adobe Targetのレコメンデーション機能では、過去のユーザーアクティビティなどのアルゴリズムにもとづいて、顧客が関心を持つ可能性のある商品やコンテンツが自動的に表示されます。 レコメンデーションにより、顧客が関心を持ちそうな商品を積極的に紹介することが可能になります。

>[!NOTE]
>
>レコメンデーションのドキュメントは、専用のセクションに移動しました。 詳しくは、[レコメンデーション](/help/main/c-recommendations/recommendations.md#concept_7556C8A4543942F2A77B13A29339C0C0)を参照してください。
