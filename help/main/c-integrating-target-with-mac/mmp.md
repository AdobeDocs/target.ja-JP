# Experience Cloud Audiences

[!DNL Adobe Experience Cloud Audiences] は、 [!DNL Adobe Experience Cloud] 各ソリューションで同じデータを使用できるように、web サイトの訪問者に関する情報を伝え、共有するソリューション。
