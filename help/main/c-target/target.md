# オーディエンス

[!DNL Adobe Target]のオーディエンスを使用して、特定のオーディエンスに対する様々なコンテンツやエクスペリエンスのターゲティングを行い、適切なメッセージを的確なオーディエンスにタイミングよく表示することで、web マーケティングを最適化します。

オーディエンスは、ターゲットを絞ったアクティビティを表示する、類似のアクティビティ参加者のグループです。 オーディエンスとは、カリフォルニア州からの新規訪問者、再訪問者、再訪問者など、同じ特徴を持つ人々のグループのことです。 訪問者がターゲットオーディエンスに当てはまる場合は、[!DNL Target] により、アクティビティ作成時に定義された条件に基づいて、そのユーザーに表示するエクスペリエンスが決定されます。

デフォルトでは、トラフィックはエクスペリエンス間で等しく分割されます。 ただし、各エクスペリエンスの目標の割合を指定できます。

オーディエンスはルールベースにすることもできます。 ルールベースのオーディエンスは、訪問者の場所、ブラウザー、オペレーティングシステム、モバイルデバイス、その他のデータなど、訪問者について収集された情報にもとづいています。

全員に同じコンテンツを表示しながら、レポートデータをグループごとに分割するには、オーディエンスの代わりにセグメントフィルターを使用します。

アクティビティが設定されたページを訪問者が訪れた場合、[!DNL Target] では、アクティビティに指定されたオーディエンスのメンバーにその訪問者が当てはまるかどうかが特定されます。 訪問者がターゲットオーディエンスのメンバーとして識別できない場合、その訪問者はデフォルトのコンテンツが表示され、そのアクティビティのレポートには含まれません。

訪問者がターゲットオーディエンスに当てはまる場合は、[!DNL Target] により、アクティビティ作成時に指定された条件に基づいて、そのユーザーに表示するエクスペリエンスが決定されます。
